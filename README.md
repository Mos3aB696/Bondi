# Bondi

**Bondi** is a fully responsive front-end website designed using **Bootstrap 5**. This project showcases a sleek, modern interface with well-organized sections, demonstrating best practices in front-end design. It is ideal for use in portfolios, small business websites, or as a template for other web projects.

## Features

- **Responsive Layout**: Adapts to various screen sizes for a seamless experience across devices (mobile, tablet, and desktop).
- **Bootstrap 5 Integration**: Utilizes Bootstrap's grid system, components, and utilities to ensure consistency and ease of customization.
- **Custom Styling**: Includes additional CSS to enhance the basic Bootstrap styling.
- **Cross-browser Compatibility**: Optimized to work smoothly on major web browsers.

## Technologies Used

- **HTML5**: For building the structure of the webpage.
- **CSS3**: For styling and layout customization.
- **Bootstrap 5**: The main framework for responsive design and UI components.

## Installation

To use this project locally:

1. **Clone the repository**:
    ```bash
    git clone https://github.com/Mos3aB696/First_Design_With_Bootstrap_5.git
    ```

2. **Navigate into the project directory**:
    ```bash
    cd First_Design_With_Bootstrap_5
    ```

3. **Open the index.html file in your browser**:
   You can directly open the `index.html` file in any modern browser to view the project locally.

    ```bash
    open index.html
    ```

This project does not require a server as it is a static website.
